This website was designed for the BiCS smart garden, which is located in the BiCS lab in Belval.
The development of this website was taken care of by a team, which consisted of a member of the BiCS team and a student of the BiCS bachelor.

What is the BiCS' smart garden ?

The BiCS' smart garden is a garden located in the BiCS lab. It is called a 'smart' garden, because it embodies smart devices, such as smart plugs, which makes it so that it is connected to the internet.

Value box

This row contains a div with a 'value box', which displays the most-recent measured values, that have been measured by the smart devices of the smart garden.
